# Create README.txt file

Human Framework is a high-level declarative language designed to describe interfaces and behaviors using human-readable keywords. It uses natural syntax, with no parentheses, no symbols, and no traditional programming structures.

Goal
Provide an accessible, universal way to define elements, properties, and user interactions without requiring programming knowledge.

Quick Example
create
name button-login
color blue
size medium
when click -> color yellow
